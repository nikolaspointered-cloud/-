# ElectroProm — простой рекламный сайт электроники

Файлы проекта:

- `index.html` — главная страница
- `styles.css` — стили сайта
- `script.js` — логика отображения товаров и корзины

Как запустить:

1. Откройте `index.html` в браузере (двойной клик или `File -> Open`).
2. Нажмите кнопку "Добавить в корзину", затем откройте корзину.

Примечания:
- Картинки берутся с `source.unsplash.com` (замена изображений возможна).
- Корзина хранится в `localStorage` браузера (ключ `cart_v1`).
